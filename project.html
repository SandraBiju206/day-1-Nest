  3. Smart Dustbin using Arduino Uno
    Built an Arduino-based smart dustbin that opens automatically when a person is detected using an IR sensor. The system also monitors the fill level of the dustbin to improve waste management efficiency. This project focuses on hygiene, automation, and smart city concepts by reducing human contact and improving waste disposal methods.

    Technologies Used:

    Arduino Uno, IR Sensor, Embedded C
